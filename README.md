# books mini project
A mini project about book searching using google apis

## MERN Stack
- MongoDB Atlas
- Express 4.18
- ReactJs 18.2
- NodeJs  17.5

## Usage
Rename the .env.example to .env and add your env variables

### Install dependencies
```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server
```
npm run server
```
